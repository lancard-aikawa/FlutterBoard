ストアスクリーンショット / 動画生成

概要

Playwright を使って Flutter Web アプリを自動撮影し、Play Store / App Store 向けの スクリーンショットや動画を生成します。

必要なツール

ツール用途必須
Playwrightブラウザ自動操作・撮影✅ 必須
ffmpegWebM → GIF / MP4 変換動画変換時のみ

手順 1 — Playwright をインストールする

Flutter プロジェクトのルートで実行します。

npm install -D playwright
npx playwright install chromium

パネル上部のバッジが Playwright ✓ になれば準備完了です。

手順 2 — ffmpeg をインストールする（任意）

GIF・MP4 変換が不要な場合はスキップできます。

winget（Windows 11 標準）

winget install Gyan.FFmpeg

Chocolatey

choco install ffmpeg

インストール後、新しいターミナルを開いて ffmpeg -version で確認してください。

手順 3 — Flutter Web アプリを起動する

撮影対象は Flutter Web ターゲット のアプリです。 --web-port でポートを固定してから起動してください。 

flutter run -d chrome --web-port=8080
ポートを固定する理由： --web-port を省略するとポートが毎回ランダムになり、 ベース URL の設定が毎回必要になります。
ログに表示される ws://127.0.0.1:xxxxx/…（VM Service）や http://127.0.0.1:9100（DevTools）は撮影対象の URL ではありません。

手順 4 — パネルを設定する

ベース URL

Flutter Web アプリの URL を入力します。 flutter run --web-port=8080 の場合は http://localhost:8080 です。

撮影ルート

撮影したい画面をルートとして追加します。
「名前」は出力ファイル名になります（例: home → home.png）。
「パス」はベース URL からの相対パスです（例: /、/settings）。

名前パス説明
home/トップ画面
settings/settings設定画面

Viewport

撮影するデバイスサイズを選択します。複数選択可能です。

プリセットCSS サイズDPR物理解像度用途
Phone360 × 80031080 × 2400Play Store スマートフォン向け
7" Tablet600 × 96021200 × 1920Play Store 7インチタブレット向け
10" Tablet800 × 128021600 × 2560Play Store 10インチタブレット向け
DPR（デバイスピクセル比）とは： CSS ピクセルに対する物理ピクセルの倍率です。DPR=3 の場合、 CSS 360px が物理 1080px に相当し、実機スマートフォンと同じ見た目でレンダリングされます。

出力オプション

オプション説明
動画 (WebM)各画面の読み込みを録画。ffmpeg があれば変換も可
GIF 変換WebM → GIF（ffmpeg 必要。ファイルサイズに注意）
MP4 変換WebM → MP4（ffmpeg 必要。Play Store プロモ動画向け）

手順 5 — 撮影する

  1. 「設定を保存」で設定を保存する
  2. Flutter Web アプリが起動していることを確認する
  3. 「撮影開始 ▶」を押す（最大 120 秒）
  4. 完了後、ギャラリーに画像が表示される
  5. 各画像の「PNG ↓」ボタンでダウンロードできる

出力先: （プロジェクト）/screenshots/store/（タイムスタンプ）/

よくあるエラー

SecurityError: Failed to execute 'replaceState' on 'History'

Flutter Web の URL Strategy 起因のエラーです。スクリーンショット自体は取得できますが、 アプリがエラー画面を表示している場合があります。

根本解決は main.dart でハッシュ形式の URL Strategy に切り替えることです。

import 'package:flutter_web_plugins/url_strategy.dart';

void main() {
  useHashUrlStrategy();
  runApp(const MyApp());
}

Playwright が見つかりません

手順 1 のインストールを行い、サーバーを再起動してから再試行してください。

タイムアウト（120秒超）

Flutter Web アプリが起動していないか、ベース URL が間違っています。 Chrome で直接 URL を開いて確認してください。
